6) 성능평가방법 확인¶

6.1) RMSLE(Root Mean Square Log Error)¶

  • RMSE(Root Mean Squared Error) 에 L(Log)가 추가됨
  • 즉, 로그값을 취하여 오차를 구하는 RMSE 방식
  • Error이므로 값이 작을 수록(0에 가까운 값이 나올 수록) 오류가 적다는 의미

[참고]RMSE (Root Mean Squared Error)¶

[참고] log(X) 보다 log1p(X) 를 선호하는 이유¶

  • log0= -무한대, log1=0 이므로 x>=0 일 경우 log1p(x)>=0이 보장됨
  • (X값이 0이상의 범위라면) -무한대를 피하여 계산오류를 방지할 수 있어 일반적으로 log1p()를 선호함

6.2) RMSLE 함수 작성¶

  • 사이킷런은 RMSLE를 제공하지 않아 직접 만들어서 사용해야 함

def rmsle(pred, y):
    log_pred = np.log1p(pred) 
    log_y = np.log1p(y)
    squared_error = (log_pred-log_y)**2   
    rmsle = np.sqrt(np.mean(squared_error))
    return rmsle

6.4) evaluate_regr() - RMSLE, RMSE, MAE를 모두 계산하는 함수¶

In [ ]:
def evaluate_regr(pred, y):
    rmsle_val = rmsle(pred, y)
    rmse_val  = rmse(pred, y)
    mae_val  = mae(pred, y)
 
    print('RMSLE: {0: .3f}, RMSE: {1: .3f}, MAE: {2: .3f}'.format(rmsle_val, rmse_val, mae_val))

def get_model_predict(model, X_train, X_test, y_train, y_test, is_expm1=False):
    model.fit(X_train, y_train)
    pred = model.predict(X_test)
    if is_expm1 :
        y_test = np.expm1(y_test)
        pred = np.expm1(pred)
    print('###', model.__class__.__name__, '###')
    evaluate_regr(pred, y_test)
