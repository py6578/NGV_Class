import seaborn as sns

import warnings as wn

y = x ** 2 + 2*x + 5.1

1) subplot으로 여러 영역의 그래프를 그리는 방법 (nrows=N, ncols=M) 일때¶

  • 방법. fig, ax = plt.subplots(nrows=N, ncols=M)

6개의 그래프를 아래와 같이 그려보자.¶

  • 단, figsize=(15,5)
In [4]:
fig, ax = plt.subplots(nrows=2, ncols=3, figsize=(15,5))

# subplot의 ax이용하여, 첫번째 그래프에 선그래프 그리기
ax[0][0].plot(x, y)

#  subplot의 ax이용하여, 두번째 그래프에 산점도 그리기
ax[0][1].scatter(x, y)

#  subplot의 ax이용하여, 세번째 그래프에 barplot 그리기
ax[0][2].bar(x, y)

#  네번째 그래프에 seaborn의 barplot 그리기 & 격자넣기 
sns.barplot(x=x,y=y,ax=ax[1][0])
ax[1][0].grid(True)

#  다번째그래프에 seaborn의 선그래프(lineplot) 그리기 & 격자넣기 
sns.lineplot(x=x,y=y,ax=ax[1][1]) 
ax[1][1].grid(True)

#  여섯번째 그래프에 seaborn의 산점도 (scatterplot) 그리기 & 격자넣기 
sns.scatterplot(x=x,y=y, ax=ax[1][2]) 
ax[1][2].grid(True)
  • 그래프 사이즈 설정 (figsize 인수설정)
    • plt.subplots(2, 3, figsize=(15,5))
