로지스틱 회귀 (Logistic Regression) 실습 - 위스콘신 유방암 데이터 예측¶

실습 목적¶

  • 종양의 크기, 모양 등 다양한 속성값을 기반으로 악성 종양인지 양성 종양인지를 분류함.
  • 스케일링 해보기 실습

실습 순서 이해¶

다음 순서로 실습을 해 본다.

  • (1) 라이브러리/모듈 import
  • (2) 위스콘신 유방암 데이터 세트 호출 (load_breast_cancer())
  • (3) 데이터를 학습데이터와 테스트 데이터로 분리
  • (4) 로지스틱 회귀분석 실시

악성(malignant) - 암o (1) 양성(benign) - 암x (0)

(2) 데이터셋 로드¶

sklearn 의 datasets는 sklearn.utils.Bunch라는 key-value 형식으로 구성되는 딕셔너리 형 타입과 유사한 구조를 가지고 있습니다.

  • 공통 키 key
    • data: 샘플 데이터, Numpy 배열로 이루어져 있습니다.
    • target: Label 데이터, Numpy 배열로 이루어져 있습니다.
    • feature_names: Feature 데이터의 이름
    • target_names: Label 데이터의 이름
    • DESCR: 데이터 셋의 설명
    • filename: 데이터 셋의 파일 저장 위치 (csv)dataset = load_breast_cancer()

  • 피쳐들만 X_feature, 타깃(목표변수)를 y_label
  • axis = 1 : 열방향,(즉, 열전체)

X_features = cancer_df.drop('target', axis=1) 
y_label = cancer_df['target']

In [ ]:

X_train , X_test, y_train , y_test = train_test_split(X_features, y_label, test_size=0.3, random_state=0)
  • 인수: x값들, y값들, test_size(테스트데이터 크기), random_state(무작위 추출할 때 매번 변하지 않게 하기 위해 설정)
